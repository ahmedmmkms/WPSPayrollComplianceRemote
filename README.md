# WPS Payroll Compliance

Multi-tenant payroll tooling targeting UAE WPS and KSA Mudad compliance, delivering SIF exports with auditability.

## MVP Scope
- GitHub Actions-driven CI/CD
- Laravel 11 + Filament 3 backend
- Multi-tenant payroll ingestion, validation, and export pipeline

## Sprint 0 Priorities
- CI/CD pipelines and DevOps foundations
- Environment topology and IaC scaffolding
- Localization and compliance requirement capture
